## Project Overview
This repository contains a structured audit of **Wireshark**, the world's leading open-source network protocol analyzer. This project explores its GPL v2 licensing, Linux integration, and contribution to the FOSS ecosystem.

## Shell Scripts Description
This project includes five bash scripts demonstrating core Linux automation concepts:
1. **script1_identity.sh**: Generates a system identity report (Kernel, Uptime, User).
2. **script2_inspector.sh**: Checks if Wireshark is installed and provides a philosophy note.
3. **script3_auditor.sh**: Audits permissions and disk usage of key system directories.
4. **script4_analyzer.sh**: Scans log files for specific keywords like "ERROR".
5. **script5_manifesto.sh**: An interactive script that generates a personalized OSS manifesto.
